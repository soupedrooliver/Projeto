# 🌑 DevLinks

Uma página pessoal estilo Linktree com modo claro e escuro, desenvolvida com HTML, CSS e JavaScript.

---

## 🚀 Sobre o Projeto

O **DevLinks** é uma aplicação moderna que reúne links pessoais em um único lugar, ideal para usar na bio de redes sociais como Instagram, LinkedIn e GitHub.

Este projeto foi desenvolvido com foco em praticar fundamentos sólidos de front-end.

---

## ✨ Funcionalidades

- 🌗 Alternância entre modo claro e escuro
- 🎨 Uso de variáveis CSS (`:root`)
- 🧠 Manipulação do DOM com JavaScript
- 🎬 Animações suaves com `transition` e `transform`
- 📱 Layout responsivo
- 🔗 Links com efeito hover
- 👤 Alteração dinâmica da foto de perfil

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Inter)
- Ionicons

---

## 💡 Conceitos Aplicados

Este projeto reforça conhecimentos em:

- Estruturação semântica em HTML
- CSS Variables
- Pseudo-classes (`:hover`)
- `classList.toggle()`
- Responsividade
- Organização de código
